/content/content

Returns the full details of an individual item of content. Every content item listed in the search results provides the url at which to get the individual content item.

Sample query

Sample query: /gdn-api/content/334667324

Parameters

format Specifies what format to return results in, the supported values are: xml and json. If this is not specified results are returned in xml format.
callback Specifies a javascript function to be invoked with the results from a query.
